Exercise 3: Traversing the DOM

This exercise about traversing from one part of the DOM to another, both up and down the source tree.

  • Replace the text in the span of class 'nextText' with the text found in the span of class 'targetText' using the 'text()' method. Start from the div of class 'sample'.
  • Starting from the span of class 'targetText', copy the text and insert it into the paragraph of class 'emptyText' by traversing.
