Exercise 9: Animate

This exercise is about animating elements on a page. Make the div of class 'target' appear when Button 1 is pressed. This animation requires the use of one of the jQuery methods in the Effects section of the jQuery API. (http://api.jquery.com/category/effects/). The second animation will fade in the div of class 'target2' and move it 200 pixels to the right. This will require the use of the 'animate' method which can accept a large number of arguements. (http://api.jquery.com/animate/).
